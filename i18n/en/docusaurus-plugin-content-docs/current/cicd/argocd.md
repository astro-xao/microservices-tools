# ArgoCD
## 介绍
ArgoCD 是一个 Kubernetes 原生的 GitOps 工具，专注于持续交付（CD）。
- **功能特点**：
    - 专注于 CD（持续交付），通过 GitOps 管理 Kubernetes 集群的应用部署。
    - 自动将 Git 仓库中的声明式配置同步到 Kubernetes 集群。
    - 提供实时监控、回滚和多环境支持。
- **适用场景**:
    - 部署 Kubernetes 应用。
    - 管理多环境（如开发、测试、生产）的配置。
    - 实现 GitOps 工作流。
